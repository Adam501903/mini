*:required
[]: optional
Commands available :
~help
help [command name]:
    -if no command was written, shows the entire manual.
    -if one single command name, shows the part of manual dedicated to the command written.
~exit
exit:
    -stops the interpreter from working.
~cd
cd <*DirName> :
    -changes current directory.
~ls:
ls [options: -l | ] [RepName] :
    -shows whats in a given directory (current directory if none mentionned)
~pwd
pwd: 
    -shows current position.
~date
date:
    -shows current date.
~touch
touch:
    -create a text file with the name.
~mkdir 
mkdir <*DirName> :
    -creates a directory with the name.
~rmdir
rmdir <*DirName> :
    -deletes the file and its content.
