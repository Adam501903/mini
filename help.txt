*:required
[]: optional
Commands available :

~help [command name] :
    -if no command was written, shows the entire manual.
    -if one single command name, shows the part of manual dedicated to the command written.
~exit : stops the interpreter from working.
~cd <*RepName> : changes current repository.


~ls [-l ]
~pwd : shows current position.
~date: shows current date.
~touch: create a text file with the name.
~mkdir <*RepName> : creates a repository with the name.
~rmdir <*RepName> : deletes the file and its content.
